About the Course Project
• This is a deliverable for the course Developing Data Products as part
of the Coursera Data Science Specialization.
• Instructions:
• Write a shiny application with associated supporting documentation. The
documentation should be thought of as whatever a user will need to get
started using your application.
• Deploy the application on Rstudio's shiny server
• Share the application link by pasting it into the provided text box
• Share your server.R and ui.R code on github
How to use the application
• Using the data provided by Galton Families dataset, we fit a linear
model to predict a child's height based on the gender and parent's
average height.
